# Syntax

( similar to ScriptScript )

Types will be "strict" so unless manually converted they won't change.
To define a variable:
```var x num = 10;```
'x' is the name of the variable an 'num' is the type. There are 5 types: 
'str', 'num', 'str', 'arr', 'bool'
